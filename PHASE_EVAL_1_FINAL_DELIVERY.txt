9. 使用方式
================================================================================

基础用法：
  python -m scripts.eval_scenario_results

自定义路径：
  python -m scripts.eval_scenario_results \
      --input my_results.csv \
      --output my_eval.csv

禁用终端打印：
  python -m scripts.eval_scenario_results --no-pretty-print

运行测试：
  pytest tests/test_eval_scenario_results.py -v

查看帮助：
  python -m scripts.eval_scenario_results --help
